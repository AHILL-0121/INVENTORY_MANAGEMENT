# Frontend Setup

## Prerequisites
- Node.js 18+
- npm or yarn

## Installation

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file with:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

3. Run the development server:
```bash
npm run dev
```

Server will start at: http://localhost:3000

## Build for Production

```bash
npm run build
npm start
```

## Default Login

After backend is running and you create an admin user, use those credentials to login.
